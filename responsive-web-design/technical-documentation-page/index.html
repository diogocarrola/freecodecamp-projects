Introduction

JavaScript is a cross-platform, object-oriented scripting language.

It is a small and lightweight language.

console.log("Hello World");

JavaScript contains a standard library of objects.

  • Client-side JavaScript
  • Server-side JavaScript
JavaScript and Java

JavaScript and Java are similar in some ways but fundamentally different.

JavaScript was influenced by Java but has different syntax.

// JavaScript variable let x = 5;
  • Java is compiled, JavaScript is interpreted
  • Java is strongly typed, JavaScript is dynamically typed
Variables

You use variables as symbolic names for values in your application.

Variable names can contain letters, digits, underscores, and dollar signs.

var x = 42; // Global variable let y = 13; // Block-scoped variable

JavaScript has three ways to declare variables:

  • var
  • let
  • const
Data Types

The latest ECMAScript standard defines eight data types:

Seven primitive data types and Object.

typeof "John" // Returns "string"
  • Boolean
  • Null
  • Undefined
  • Number
  • BigInt
  • String
  • Symbol
  • Object
Functions

A function is a block of code designed to perform a particular task.

Functions are executed when they are called.

function myFunction(p1, p2) { return p1 * p2; }

Functions can be defined in several ways:

  • Function declarations
  • Function expressions
  • Arrow functions
